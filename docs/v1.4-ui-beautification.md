# StudyTimer v1.4 — UI 美化记录

## 设计语言
- **风格**: iOS 玻璃拟态 · 毛玻璃特效 · 极简排版
- **主色**: 雾蓝 `#5B9BD5`（从 `#6B9FC7` 微调）
- **页面底色**: 暖奶油 `#F8F5F0`
- **字体**: `sans-serif-medium` · 四级文字层次 · 字间距 0.02-0.08
- **卡片**: 高透明度玻璃 + 0.5dp 渐变描边 + 柔和阴影 + 顶部高光条

## 核心改动

### 1. 色彩体系 (colors.xml)
- 主色 `#5B9FC7` → `#5B9BD5` 更明亮
- 新增四级文字层次：`text_quaternary`
- 新增分隔线色：`divider`, `divider_light`
- 新增阴影色：`shadow_soft`, `shadow_medium`
- 玻璃拟态色更精细：`glass_fill_strong`, `glass_stroke_subtle`

### 2. 卡片系统
- `card_glass.xml` — 重写：渐变描边 + 柔和阴影 + 顶部高光条
- `card_elevated.xml` — 新增：双层阴影 + 强玻璃感

### 3. 毛玻璃效果 (DynamicBackgroundView.kt)
- Android 12+ 真毛模糊：`RenderEffect.createBlurEffect(25px)`
- 漂移速度减慢（12秒周期，更柔和）
- 三色渐变更明显

### 4. 底部导航 (activity_main.xml)
- 改为纯图标模式（`labelVisibilityMode="unlabeled"`）
- 使用 `card_elevated` 浮起效果
- 圆角胶囊选中指示器
- 新增 `nav_icon_color.xml` 颜色选择器

### 5. 排版优化
- 标题 32sp + `sans-serif-medium` + 字间距 0.02
- 小标题 13sp + 全大写 + 字间距 0.08（iOS 风格 section header）
- 按钮高度 56dp + 渐变背景
- 输入框字体 `sans-serif-medium`

### 6. 我的页面 (fragment_profile.xml)
- iOS 风格分组列表（每组一个卡片）
- 组内用 0.5dp 分隔线
- 箭头用 `›` 替代 `>`（更 iOS）
- 点击态添加 `selectableItemBackground`

### 7. 统计页面 (fragment_stats.xml)
- 总时长卡片用 `card_elevated`
- 更大留白（24dp 水平内边距）
- 标题字体统一

### 8. 图表颜色更新
- CircularTimerView: 弧色 `#5B9BD5` → `#7B6EC8` → `#A06CC8`
- PieChartView: 饼图色统一为新主色
- LineChartView: 折线 + 填充 + 标签色全部更新

### 9. 按钮样式
- `btn_pill_primary.xml` — 渐变蓝→靛紫 + 柔和阴影

## 兼容性
- 真毛模糊仅 Android 12+ 生效，低版本回退到普通绘制
- 所有旧颜色引用已全局替换

## 已知限制
- 无法本地编译（无 Java 环境），需在 Android Studio 中验证
